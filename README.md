# ThinMP_KMP

This app is a simple music player for Android and iOS.

## Environments

### host

* KMP
* CMP

### Android

* Android Studio | 2025.2.2 Patch 1
* Kotlin
* Gradle
* minSdkVersion 33
* targetSdkVersion 36
* Google Pixel 4a (Android Version 13)
* Google Pixel 7 Pro (Android Version 16)
* Google Pixel Tablet (Android Version 16)

### iOS

* Xcode 26.0
* Swift
* iOS Deployment Target 16
* iPhone 12 mini (iOS 16)
* iPhone 14 Plus (iOS 26)
* iPad Pro (6th generation, iPadOS 18)
